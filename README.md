# GS2-SERS - Eficiência Energética Escritório

Projeto de análise de consumo de energia em escritório.

## Estrutura do Projeto

```
gs2-sers-eficiencia-energetica-escritorio/
├── dados/
│   └── consumo_energia_escritorio.csv
├── src/
│   └── analise_consumo.py
├── graficos/
│   ├── consumo_diario.png
│   └── economia_estimativa.png
├── README.md
```

## Descrição

Este projeto analisa o consumo de energia em escritório para identificar oportunidades de eficiência energética.

## Uso

Execute o script de análise:

```bash
python src/analise_consumo.py
```

## Requisitos

- Python 3.x
- pandas
- matplotlib
